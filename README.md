# Koi Pond 🐟

An interactive, highly aesthetic Koi Pond. Move your mouse to create ripples and click to feed the fish!

## 🎥 Preview

[![Koi Pond Wallpaper Demo](https://img.youtube.com/vi/0NkA6I73C5Q/maxresdefault.jpg)](https://www.youtube.com/watch?v=0NkA6I73C5Q)

## 🌟 Features

- **Interactive Environment:** Move your cursor over the water to create dynamic mathematical ripples.
- **Feed the Koi:** Click anywhere in the pond to drop food. The fish will intelligently seek it out!
- **Procedural Animation:** Fish are drawn and animated procedurally using segment-based kinematics for incredibly realistic movement.
- **Aesthetic Caustics:** Beautiful, procedurally generated light caustics overlay the pond.
- **Highly Customizable:** Fully integrated with Wallpaper Engine properties:
  - Tweak Koi count, speed, and size.
  - Adjust the water color hue to match your setup.
  - Change fish themes (_Traditional, Neon, Monochrome_).
  - Toggle caustics and feeding.
  - Control your ripple strength.

## 🚀 How to Use

### Option 1: Steam Workshop (Recommended)

The easiest way to install this wallpaper is directly through the Steam Workshop if you own Wallpaper Engine.

1. Go to the [Steam Workshop Page](https://steamcommunity.com/sharedfiles/filedetails/?id=3692215641).
2. Click the green **Subscribe** button.
3. Open **Wallpaper Engine**, and the Koi Pond wallpaper will automatically download and appear in your installed wallpapers list!

### Option 2: Wallpaper Engine (Local File)

1. Open **Wallpaper Engine**.
2. Click **Open Wallpaper** at the bottom left.
3. Select **Open offline wallpaper (animated)**.
4. Navigate to this folder and select the `project.json` or `index.html` file.
5. Customize the settings via the properties panel on the right!

### Option 3: Free Alternative (Lively Wallpaper)

Don't have or want to purchase Wallpaper Engine? You can still use this beautiful wallpaper completely for free using **Lively Wallpaper**!

1. Download and install [Lively Wallpaper](https://rocksdanister.github.io/lively/).
2. Open the Lively application and click the **Add Wallpaper** (`+`) button at the top.
3. Simply drag and drop the `index.html` file (or select the whole folder) into the Lively window.
4. Apply it as your background and enjoy!

## 🛠️ Built With

- **Vanilla JavaScript:** Utilizing `CanvasRenderingContext2D` for high-performance visual rendering without heavy dependencies.
- **HTML5 & CSS3:** For structuring and creating an atmospheric vignette effect.
- **Math:** Custom inverse kinematics and physics for fluid ripple equations and Koi swimming.
